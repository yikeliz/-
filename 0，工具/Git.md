## 个人总结：

1,如果在线上修改过代码，那么你得先同步到本地，然后，在进行上传。

```
git pull --rebase origin master  先同步到本地
git push -u origin master  把commit代码上传



```



2，其他常用的指令

```
git status 查看状态
git add -A 增加文件
git commit -m "提交信息"
git log 提交记录

```
